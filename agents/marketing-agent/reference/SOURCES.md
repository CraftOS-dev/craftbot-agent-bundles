## Notes on "authored from synthesis"

Several sections include composition synthesis on top of the referenced material:

- **Three opening convictions in soul.md** — synthesizes three load-bearing rules from three different reference agents into a memorable triad. Each conviction comes from a reference; the triad framing is composed.
- **Content brief template** — synthesizes elements present across reference agents (audience, intent, format, distribution, CTA, brand voice notes) into a single template. Sections mapping to the reference agents are noted in the role.md inline.
- **Marketing technology categories** — names specific ESPs from `msitarzewski-email-strategist.md` plus industry-standard categories (CRM, automation, analytics, etc.). Listing common tools is industry knowledge, not invented.
- **Campaign brief template** — synthesized from campaign management sections of `voltagent-content-marketer.md`. Format is composed.
- **AI-slop catch list** — `voltagent-content-quality-editor.md` provides the categorization; the specific phrases-to-strip list is expanded with common industry-known AI-text patterns (this is a curation move, not invention).
- **First-conversation PROACTIVE.md self-init** — standard pattern from PROGRESS.md decision #3 with marketing-specific routine questions.

No domain claims, performance benchmarks, or compliance dates were invented. All benchmarks come from the references (especially `msitarzewski-email-strategist.md` which is exceptionally rich with 2024-2025 dates).

## How to update this agent

1. Re-fetch source files listed in `reference/INVENTORY.md` and overwrite `reference/agents/*.md` in place
2. Diff against previous versions to see what changed
3. Update corresponding sections of `soul.md` and `role.md`
4. Update this `SOURCES.md` if section names or source URLs changed
5. Re-run `build.py` to regenerate `dist/marketing-agent.craftbot`
