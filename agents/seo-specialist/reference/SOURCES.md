- **JS-rendering audit decision tree** — synthesizes Screaming Frog JS-mode docs + Search Console URL Inspection docs + Lighthouse CI docs into a single decision tree. Each row traces to a vendor doc.
- **E-E-A-T scoring rubric** — synthesizes Google's E-E-A-T guidance (Search Central helpful content docs) + `msitarzewski-marketing-seo-specialist.md` E-E-A-T section + Knowledge Graph API entity verification into a 10-point rubric. The 10-point structure is composed; each point traces to a Google source.
- **Antipattern catalog** — synthesizes documented Google penalties + Spam Policies + April 2024 disavow update + AMP deprecation mid-2025 into BAD/GOOD pairs. Each antipattern traces to a Google source.
- **Schema deep templates (nested @graph)** — synthesizes Schema.org per-type docs into a single nested @graph pattern. The nesting structure is composed; each `@type` block traces to Schema.org docs.
- **AEO content optimization patterns (direct-answer block + entity-richness + source authority + structured data depth)** — synthesizes Frase AEO blog + Surfer GEO blog + LLM citation behavior research into a single AEO content pattern. The 4-element framework is composed; each element traces to an AEO vendor source.
- **First-conversation PROACTIVE.md self-init** — standard pattern from METHODOLOGY.md with deep-SEO routine questions per per-agent prompt (primary CMS / stack, main SEO platform, current rankings tracking cadence).

2. Diff against previous versions to see what changed

5. Re-run `python verify.py seo-specialist` to confirm structure intact
