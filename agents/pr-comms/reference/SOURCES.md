- **role.md contact tree template** — PR.co playbook + industry standard timing conventions.

- **role.md Show HN launch playbook** — community practice synthesis (no single canonical doc); reflects HN community norms for product launches.

## Refreshing from upstream

When SOTA tools change (e.g., new wire service API, new SME platform, new analyst firm methodology):

1. Update the relevant skill pack(s) in `agents/pr-comms/skills/<name>/SKILL.md` (Round 2 artifacts).
2. Update the SOTA sources table above.
3. Update `reference/SOTA_USE_CASES.md` confidence ratings if applicable.
4. Re-run `python verify.py pr-comms` to confirm structure intact.
5. Re-build: `python build.py pr-comms` produces a fresh `.craftbot`.
