# Data Storytelling

Convert data into persuasive narratives for business audiences. Framework emphasizes **three essential components**: evidence, meaning, and visual clarity.

## When to Use This Skill

- Communicating with non-technical stakeholders
- Creating executive reports
- Building investor presentations
- Translating dashboard insights into actionable narratives

## Key Structure

**Setup → Conflict → Resolution**

Establish context first, then present the problem, then propose solutions.

## Central Principle

**Start with the "so what".** Lead with the impact, not with raw numbers or methodology.

## Three Essential Elements

| Element | Role | Content |
|---------|------|---------|
| Data | Evidence | Metrics and trends |
| Narrative | Meaning-making | Context and implications |
| Visuals | Comprehension | Charts and diagrams |

## Critical Guidance

- **Don't data dump.** Effective storytelling requires ruthless curation.
- **Don't bury the insight.** Frontload key findings before the technical explanation.
