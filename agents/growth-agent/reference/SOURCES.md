## Notes on authored-from-synthesis

Several sections include composition synthesis on top of referenced material:

- **Three opening convictions in soul.md** — synthesizes three load-bearing rules from three different reference materials into a memorable triad. Each conviction traces to a source.
- **5 loop types description** — VoltAgent `growth-loops.md` provides the 5-type taxonomy; description is direct.
- **Retention curve shape interpretation (4 shapes + prescribed fix)** — Amplitude cohort retention guide provides shape categorization; "fix per shape" mapping is composed from Reforge retention frameworks + Stackmatix PLG handbook.
- **Antipattern catalog** — 7 antipatterns are composed from common growth-pitfalls documented across reference material (no single canonical source; this is industry knowledge curation).
- **Decision matrices (onboarding, in-app messaging, referral, loyalty)** — composed from reference material's pros/cons + 2026 pricing comparison sources.
- **Communication style targets** — composed from msitarzewski + Brian Balfour clarity rules; specific phrasings ("Day 30 retention is 12%...") are example-based.
- **First-conversation PROACTIVE.md self-init** — standard pattern from PROGRESS.md decision #3 with growth-specific routine questions (motion / analytics-tool / activation-rate).

No domain claims, performance benchmarks, or compliance dates were invented. All benchmarks come from web-research sources cited (Userpilot 2024 TTV benchmark, B2B SaaS 37.5% activation avg, Intercom Fin 50-70% auto-resolve, PLG grows 50% faster, etc.).

## How to update this agent

1. Re-run web search on cited URLs (especially Statsig MCP, GrowthBook MCP, PostHog MCP, Amplitude MCP — these evolve quarterly)
2. Diff against this SOURCES.md to identify which sections need refresh
3. Update corresponding sections of `soul.md`, `role.md`, `reference/SOTA_USE_CASES.md`
4. Re-run `python verify.py growth-agent` to confirm structure intact
5. Re-build: `python build.py growth-agent` produces a fresh `.craftbot`
